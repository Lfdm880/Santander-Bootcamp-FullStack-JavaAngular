Id vs Class

Id= sao precedidos por um #(usado para identificar um unico elemento)
class= sao precedidos por um .(usado para identificar mais de um unico elemento)

margin: espaçamento entre elementos
border: circundam o padding e o conteúdo
	podemos alterar a largura e a cor
padding: espaçamento entre a borda e o conteúdo
content: o que o bloco representa

border-radius: altera os cantos da borda

é uma boa prática manter um padrão de direção entre as margens do conteúdo. 
Se eu estilo com Margin-bottom, todos os elements devem ser estilizados assim

list-style-type: estiliza as listas
