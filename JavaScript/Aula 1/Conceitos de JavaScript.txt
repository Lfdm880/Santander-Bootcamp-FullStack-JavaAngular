Declarar:

var - escopo global e local, se nao tiver valor incial será um Null
let - escopo local de bloco, se nao tiver valor incial será um Null
const - escopo local de bloco, nao pode ter seu valor alterado e obrigatoriamente precisa de um valor iniciado

escopo global- quando a variável é declarada em qualquer bloco (função), sua visibilidade fica disponível em todo código
escopo local - quando a variável é declarada dentro de um bloco, sua visibiliade pode ficar disponível ou nao

Regras:
Nao inciar com numero
Nao usar espaços
Nao usar palavras reservadas

Atribuição - usando '='
Comparação - usando '==' ; retorna um valor boleano
Comparação identica '===' compara o valor e o tipo.

Operadores relacionais
 > maior qualquer
 < menor que;
 >= maior ou igual a;

Operadores Lógicos

&& - "e" considera todos os valores sejam true;
|| - "ou" considera qualquer valor seja verdade
! - "nao" inver o valor de true para false e vice-versa
