Arrays ou vetores sao um tipo de lista, ou matriz de variáveis
onde cada variável possui um índice

Sintax:

let array = [];

manipulando Arrays
forEach() -intera um array;
push() - add intem no final do array
pop() - remove item no fnal do array
shift() remove item no inicio do array
unshift() - add item no inicio do array
indexOf()- retorna o indece de um valor
splice() - remove ou substitui um item pelo indice
slice() - retorna uma parte de um array existente

Objetos
Dados que possuem proriedades e valores que definem suas características.
Deve ser declarado entre chaves
